ANOVA

Finding the best-fit model

Now we fit ANOVA and look for the best model based on some criteria. Here the criteria is the AIC value. The lower the value, the better the model.

#install.packages(c( "ggpubr", "tidyverse", "broom", "AICcmodavg"))
library("sjPlot")
library(ggpubr)
library(tidyverse)
library(broom)
library(AICcmodavg)


model=NULL
vars = c("type_purchase","type_property","rooms_count","parkings_count","district")

for(i in 1:length(vars)){
  xx = combn(vars,i)
  if(is.null(dim(xx))){
    fla = paste("price ~ ", paste(xx, collapse="+"))
    model[[length(model)+1]]=aov(as.formula(fla),data=data)
  } else {
    for(j in 1:dim(xx)[2]){
      fla = paste("price ~ ", paste(xx[1:dim(xx)[1],j], collapse="+"))
      model[[length(model)+1]]=aov(as.formula(fla),data=data) 
    }
  }
}

Regression

Fitting regression model

Here, we fit a regression model to the data in order to use our preferences, based on some variables. Here the variables are latitude, longitude, type of the property, type of the purchase, room counts and parking spots. In our model here, we considered variables latitude and longitude as certain variables to be included in the model and checked whether other variables can improve the model and worth keeping.

The following table consist of estimates for regression parameters and confidence intervals, t-statistics and p-values. All variables included in this model are statistically significant as the p-value is really small. For categorical variables, dummy variables are created to account for different levels of these categorical variables.

#########################regression

# create a NULL vector called model so we have something to add our layers to
model=NULL

# create a vector of the dataframe column names used to build the formula
vars = c("type_property","type_purchase","rooms_count","parkings_count")
# remove variable names you dont want to use (at least
# the response variable (if its in the first column)

# the combn function will run every different combination of variables and then run the glm
for(i in 1:length(vars)){
  xx = combn(vars,i)
  if(is.null(dim(xx))){
    fla = paste("price ~ latitude + longitude +", paste(xx, collapse="+"))
    model[[length(model)+1]]=glm(as.formula(fla),data=data)
  } else {
    for(j in 1:dim(xx)[2]){
      fla = paste("price ~ latitude + longitude +", paste(xx[1:dim(xx)[1],j], collapse="+"))
      model[[length(model)+1]]=glm(as.formula(fla),data=data) 
    }
  }
}
